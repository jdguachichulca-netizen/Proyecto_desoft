MÓDULO HOLOGRÁFICO

REPORTE DE ESTADO:
Los circuitos de voz de W-BIT estan "sobre cargados" por usar el método antiguo de sumar textos (+).

Solución: Activar el Proyector Holográfico. Esta tecnología inyecta datos directamente dentro del texto usando el código: ${...}.

MANUAL: TEMPLATE LITERALS

ANTIGUO (Lento) "Hola " + nombre
⬇️ NUEVO (Holográfico) ⬇️
MODERNO (Rápido) `Hola ${nombre}`

OBJETIVO:

  1. Crea: let sector = "Alfa";
  2. Crea: let amenaza = 0;
  3. Muestra en consola:
  4. `Reporte: Sector ${sector}. Amenaza: ${amenaza}`
